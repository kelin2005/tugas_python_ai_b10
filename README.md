# tugas_python_ai_b10
# Kesimpulan dari Tugas Python (Tugas 4, 5, dan 6)

## Deskripsi
Repository ini berisi rangkuman dan implementasi pembelajaran Python yang mencakup:
- Data Structures
- Function & Class
- Modules, File I/O, dan OOP Sederhana

---

## Tugas 4: Python Data Structures
Pada tugas 4, memperlajari berbagai struktur data dasar dalam Python, yaitu:
- **List** → Menyimpan data berurutan dan dapat diubah
- **Tuple** → Mirip list tetapi bersifat immutable
- **Dictionary** → Menyimpan data dalam bentuk key-value
- **Set** → Menyimpan data unik tanpa duplikasi

### Kesimpulan:
Struktur data sangat penting karena untuk mengelola dan menyimpan data secara efisien dan teratur.  
Setiap struktur memiliki kelebihan masing-masing tergantung kebutuhan program.

---

## Tugas 5: Python Function and Class
Pada tugas 5, mempelajari:
- **Function (Fungsi)** → Untuk membuat kode lebih modular dan reusable
- **Class & Object (OOP dasar)** → Untuk merepresentasikan data dan perilaku dalam bentuk objek

### Kesimpulan:
Penggunaan function dan class membantu dalam:
- Mengorganisir kode
- Mengurangi pengulangan
- Membuat program lebih terstruktur dan mudah dipahami

---

## Tugas 6: Python Modules, File I/O, & OOP Sederhana
Pada tugas 6, mempelajari:
- **Modules (NumPy & pandas)** → Untuk pengolahan data dan statistik
- **File I/O** → Membaca dan menulis file (.txt)
- **OOP Sederhana** → Membuat class `GradeBook` untuk mengelola data nilai

### Kesimpulan:
Penggunaan library seperti NumPy dan pandas sangat membantu dalam:
- Mengolah data secara cepat dan efisien
- Melakukan analisis statistik

File I/O memungkinkan program untuk:
- Menyimpan hasil pengolahan data
- Membuat laporan otomatis

OOP membantu dalam:
- Mengelompokkan fungsi dan data dalam satu struktur
- Membuat program lebih scalable dan reusable

---

## Kesimpulan Akhir
Dari ketiga tugas diatas, dapat disimpulkan bahwa:
- Python menyediakan berbagai fitur yang powerful untuk pengolahan data
- Struktur data, function, dan OOP saling melengkapi dalam pembuatan program
- Penggunaan module eksternal mempercepat proses pengembangan
- File I/O memungkinkan integrasi program dengan penyimpanan data

Secara keseluruhan, ketiga tugas ini membantu memahami dasar pemrograman Python dari:
**pengolahan data → pembuatan fungsi → implementasi OOP → hingga penyimpanan data**
